# Nodio 0.5.0 — Release Candidate 1

**Release date:** April 2026

Version **0.5.0** is **Release Candidate 1 (RC1)** on the path to a stable release. It brings together the current feature set in a day-to-day-ready form; feedback helps polish the last details before the final release.

## What this means for you

- **Local-first:** Your projects stay **on your machine** — you control backups and sharing.
- **Board & linking:** Notes as cards, **typed connections**, **groups**, **saved views**, and a **graph lens** to focus without changing your content.
- **Inbox & list:** Notes **off-board** in the **inbox**; **table view** for large collections.
- **Write & read:** Comfortable **editor** with **Markdown**, links to other notes, **read mode**, **version history**, comments, and optional **web clipper** for embedded pages.
- **Kanban & dashboard:** **Workflow boards** per project plus **metrics and overview** on the project dashboard.
- **Export & import:** **PDF**, board images, structured export, **Markdown import**, and more.
- **Day-to-day safety:** **Automatic project backups**, restore from the project list, **audit log** in the project.
- **Workflow:** **Command palette**, customizable **shortcuts**, in-app **help**, **light/dark** UI, **German/English**.

## Inbox (quick capture) — virtualization & viewport

The **inbox drawer** (toolbar inbox icon) now uses the same performance pattern as **canvas table list mode**: **virtualized** lists and card grids (`react-virtuoso`), so only the visible slice is mounted in the DOM. A **count line** and optional **viewport line** (e.g. rows or items **a–b of c**) appear above the content; the tooltip explains that all matching notes stay **loaded from the local project** — virtualization affects **rendering**, not SQLite paging. **List** mode virtualizes the full outline; **Cards → Notes** uses a responsive **grid**; **Cards → Groups** virtualizes group cards; **group sections** virtualize large per-section note grids using the panel’s scroll parent where it helps.

## Maturity note

As a **release candidate**, this build may still receive small fixes before the “final” release. If you notice anything, it’s a great time to send feedback to the publisher.
